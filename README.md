# BD27819-SistemaBanco

Sistema bancario desarrollado con Node.js (Backend) y React + Vite (Frontend).

## Requisitos Previos

- Node.js (versión 14 o superior)
- npm

## Instalación y Ejecución

### Backend

1. Navegar a la carpeta del backend:

```bash
cd BancoPichincha/banco_pichincha_back
```

2. Instalar las dependencias:

```bash
npm install
```

3. Ejecutar el servidor:

```bash
npm start
```

El backend estará corriendo por defecto en el puerto configurado.

### Frontend

1. Navegar a la carpeta del frontend:

```bash
cd BancoPichincha/banco_pichincha_vite
```

2. Instalar las dependencias:

```bash
npm install
```

3. Ejecutar la aplicación:

```bash
npm run dev
```
